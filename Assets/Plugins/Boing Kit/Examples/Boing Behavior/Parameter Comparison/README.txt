Example: 
Parameter Comparison

This file contains instructions on how to set up effects demonstrated in this example.
For a general overview of Boing Kit, please refer to README.txt under Boing Kit's root folder.


Instructions:

Create a series of diamonds.

Add a script that pops an object back and forth to each diamond (Oscillator.cs).

Add a Boing Behavior component to each diamond.

Give each Boing Behavior component a different combinations of parameters.

Compare effects on each diamond.
