Example: 
Bush Field

This file contains instructions on how to set up effects demonstrated in this example.
For a general overview of Boing Kit, please refer to README.txt under Boing Kit's root folder.


Instructions:

All the logic in this example is contained in the script BushFieldReactorMain.cs.

Create spheres that move along circular paths.

Add a Boing Effector component to each of the sphere.

Create many instances of bushes and flowers; scatter them throughout the scene.

Add a Boing Reactor component to each of the bushes and flowers.

The spheres should push the bushes and flowers around as they move.
